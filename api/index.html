Lobstream

Real-time Bluesky posts visualized as text rain. An internet art piece at lobstream.com.

Data Source

Lobstream connects directly to the Bluesky Jetstream WebSocket. No API key required.

Endpoint: wss://jetstream2.us-east.bsky.network/subscribe
Param:    ?wantedCollections=app.bsky.feed.post
Auth:     None

Post event structure:
{
  "kind": "commit",
  "commit": {
    "operation": "create",
    "collection": "app.bsky.feed.post",
    "record": {
      "text": "Post content here",
      "createdAt": "2024-09-09T19:46:02.102Z"
    }
  }
}

Connect directly to the Jetstream WebSocket to receive the same real-time Bluesky post stream that powers Lobstream.
